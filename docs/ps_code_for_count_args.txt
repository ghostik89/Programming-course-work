/*!Подсчет количества аргументов в функции
	\param[in]: StringOfCode - строка кода, в которой подсчитываются аргументы
	\return: количество аргументов в функции, -1 - если это не функция, а переменная
*/
int CountArgs(const char StringOfCode[MAX_LENGTH]);
==========================================================================
Изначально считаем, что функция имеет один аргумент
Изначально считаем, что между круглыми скобками нет белых разделителей
1.Подсчет аргументов
если между скобок есть пространство
	для каждого символа между скобок
		если текущий символ - запятая, то увеличить количество аргументов
		если текущий символ - белый разделитель, то увеличить счетчик разделителей
если  пространство между скобками нет или полностью состоит их белых разделителей, то обнулить счетчик аргументов
если в строка кода - объявление переменной, то количество аргументов равно -1
2. Вернуть количество аргументов
